# Moni

>
> Simple loan disbursing app with aggressive repayment reminders
>
> **Important**: This app is for demonstration purposes only

### Features Used

- Messaging
    - SMS
    - USSD
    - Voice

- Payment
    - Purse
    - Mobile Payment

- State and Identity
    - Metadata
    - App Data
    - Reminders

### Usage

```sh
npm install
# fill in .env
npm start
```
